# LLM Usage

## Tool Used

Claude Code

## How I Used It

I used Claude to help implement the skeleton code into a interactive visualization. I used Claude primarily to help write D3/React code for specific visual components that would have taken a long time to implement from scratch.


## List of Prompts Used

1. "어떻게 개발 진행하면 좋을지 단계별로 계획세워줘"  
    (Assignment overview + screenshots provided)
    → Asked for a step-by-step development plan for the assignment.

2. "이 내용 한국어로 번역해서 보내주고, 실행 명령어 보내줘"  
   → Asked to translate the README instructions to Korean and provide the run command.

3. Screenshot of overlapping labels provided
   "이 부분에 보면 글자가 겹치는데 이거 개선해줘."  
   → Asked to fix overlapping Class badge / "Labeled as X" text, and update the page title.
